Sistema completo de **Delivery inspirado no iFood**, desenvolvido como projeto acadêmico da disciplina **Desenvolvimento de Sistemas de Informação**.

Aplicação web desenvolvida com **React e Vite**, com foco na simulação de pedidos online, gerenciamento de carrinho e fluxo de pagamento.

---

## 📌 Sobre o projeto

O **PriPriDelivery** é uma aplicação frontend que simula uma plataforma de delivery, permitindo que o usuário:

* Navegue por produtos/restaurantes
* Adicione itens ao carrinho
* Visualize o resumo do pedido
* Realize o processo de pagamento

O projeto foi desenvolvido com foco em práticas modernas de desenvolvimento frontend utilizando React.

---

## 🚀 Funcionalidades

✔ Listagem de produtos
✔ Adição de itens ao carrinho
✔ Controle de quantidade de produtos
✔ Remoção de itens do carrinho
✔ Resumo do pedido
✔ Tela de pagamento
✔ Integração com serviços externos (em desenvolvimento)

## 🚀 Como rodar o projeto

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/mauricio003/PriPriDelivery.git
```

### 2️⃣ Acessar a pasta

```bash
cd iFood-Pripridelivery
```

### 3️⃣ Instalar dependências

```bash
npm install
```

### 4️⃣ Executar o projeto

```bash
npm run dev
```

📍 Aplicação disponível em:

```
http://localhost:5173
```
